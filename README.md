# fyp1101-crud-personalizado
# SISTEMA DE AGREGADO DE POKEMON

Breve descripción de lo que hace tu programa y de qué trata. 

## Funcionalidades (CRUD)
Este proyecto incluye las siguientes funcionalidades a través de un menú interactivo:
- Agregar_pokemon: permite al usuario agregar pokemon,tipo,nivel y poder 
- mostrar_pokemon: Muestra la lista completa de los pokemon agregados.
- actualizar_pokemon: Modifica los datos de el pokemon existente buscando por su nombre/ID.
- elimina_pokemon: Elimina el pokemon del sistema.

## Requisitos Técnicos Aplicados
- Uso de listas y diccionarios.
- Uso de funciones documentadas.
- Validaciones y manejo de excepciones (`try/except`).
