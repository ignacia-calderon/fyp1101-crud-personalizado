## Funcionalidades (CRUD)
Este proyecto incluye las siguientes funcionalidades a través de un menú interactivo:
- Agregar_pokemon: permite al usuario agregar pokemon,tipo,nivel y poder 
- mostrar_pokemon: Muestra la lista completa de los pokemon agregados.
- actualizar_pokemon: Modifica los datos de el pokemon existente buscando por su nombre/ID.
- elimina_pokemon: Elimina el pokemon del sistema.

## Requisitos Técnicos Aplicados
- Uso de listas y diccionarios.:utilice el diccionario para guardar los pokemones,tipo,nivel y poder sirve para crear un pokemon
- Uso de funciones documentadas. :el agregar_pokemon funciona comoo lista y el comando append para agregar y remove para eliminar 
- Validaciones y manejo de excepciones (`try/except`).usamos para la validacion de datos while lo usamos en el menu  y tambien en actualizar pokemon para validar que todo sea correcto
